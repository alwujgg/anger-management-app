### 第一次使用

2. **了解界面**

## 🎯 主要功能使用

### 1. 快速记录情绪

**何时使用**：当您感到愤怒或不适时

**操作步骤**：
1. 点击首页的"快速记录情绪"按钮
2. 观察表情变化，选择符合您当前状态的评分
   - 1-3分：😊 心情平静
   - 4-5分：😕 略有不适
   - 6-7分：😡 明显愤怒
   - 8-10分：💢 极度愤怒
3. （可选）输入是什么让您生气
4. 点击"保存"按钮

**小提示**：
- 评分时滑块有触觉反馈
- 颜色会随评分变化（绿→黄→橙→红）
- 如果评分≥8分，会收到缓解建议通知

### 2. 查看情绪趋势

**何时使用**：了解自己的情绪变化模式

**操作步骤**：
1. 点击底部导航栏的"趋势"
2. 查看7天折线图
   - 横轴：日期
   - 纵轴：愤怒值（0-10）
3. 查看统计摘要
   - 记录次数
   - 平均愤怒值
   - 最高愤怒值

**数据解读**：
- **上升趋势**：最近情绪波动较大，建议多使用缓解工具
- **下降趋势**：情绪管理效果良好，继续保持
- **平稳低位**：情绪控制理想状态
- **波动较大**：识别高峰时段，找出规律

### 3. 使用呼吸练习

**何时使用**：感到愤怒、焦虑或需要快速冷静时

**操作步骤**：
1. 点击底部导航栏的"工具"
2. 选择"呼吸练习"
3. 点击"开始练习"
4. 跟随屏幕指示进行呼吸：
   - **吸气4秒**：深深吸气，让腹部鼓起
   - **屏息7秒**：屏住呼吸，保持放松
   - **呼气8秒**：缓慢呼气，完全放松
5. 重复4个周期（约1分20秒）

**最佳实践**：
- 找一个安静的地方
- 坐下或躺下，保持舒适姿势
- 专注于呼吸和屏幕动画
- 完成后静坐片刻，感受身体变化

**科学依据**：
4-7-8呼吸法由安德鲁·威尔博士推荐，能够：
- 激活副交感神经系统
- 降低心率和血压
- 减轻焦虑和愤怒
- 改善睡眠质量

### 4. 管理记录

**查看历史记录**：
- 首页显示最近20条记录
- 每条记录显示：评分、触发原因、时间

**删除记录**：
1. 在记录列表中，向左滑动记录卡片
2. 点击"确认删除"
3. 记录将被永久删除

**注意**：删除操作不可恢复，请谨慎操作

### 5. 数据导出

**何时使用**：备份数据或分析数据

**操作步骤**：
1. 进入"设置"页面
2. 点击"导出数据"
3. 数据将保存为JSON文件到设备存储
4. 文件路径会显示在提示中

**数据格式示例**：
```json
[
  {
    "id": "uuid-xxxxx",

    "intensityScore": 7,
    "trigger": "交通堵塞",
    "tags": null,
    "notes": null
  }
```

## 💡 使用技巧

### 情绪记录技巧

1. **即时记录**
   - 在情绪发生时立即记录
   - 避免事后回忆造成偏差

2. **详细描述触发原因**
   - 具体事件：不要只写"工作"，而是"客户投诉"
   - 包含情境：时间、地点、相关人员

3. **保持连续性**
   - 每天至少记录一次
   - 即使没有愤怒情绪也可记录低分值

### 数据分析技巧

1. **识别模式**
   - 一周查看一次趋势图
   - 找出高峰时段（如每周一早上）
   - 识别重复触发因素

2. **设定目标**
   - 本周平均分比上周降低1分
   - 高分值（≥8分）次数减少

3. **验证效果**
   - 使用呼吸练习后记录评分变化
   - 对比练习前后的平均分

### 呼吸练习技巧

1. **选择时机**
   - **预防性使用**：每天固定时间练习（如睡前）
   - **应急使用**：感到愤怒时立即使用

2. **环境准备**
   - 关闭其他应用通知
   - 调暗屏幕亮度
   - 戴上耳机（即使没有声音）

3. **进阶练习**
   - 初期：每天1次（4周期）
   - 进阶：每天2-3次
   - 配合正念：专注当下，不评判情绪

## ⚠️ 常见问题

### Q1: 数据会丢失吗？
**A**: 所有数据存储在本地设备，不会自动上传云端。建议定期导出数据备份。如果卸载应用或清除数据，记录会丢失。

### Q2: 评分8-10分时收到通知，如何关闭？
**A**: 当前版本通知功能固定开启，用于您的情绪安全。未来版本会增加通知设置选项。

### Q3: 可以修改已保存的记录吗？
**A**: 当前MVP版本不支持编辑记录，只能删除后重新添加。后续版本会添加编辑功能。

### Q4: 图表为什么没有显示数据？
**A**: 
- 确保至少有1条记录
- 7天趋势图需要记录在过去7天内
- 当天没有记录时，该天数据点为0

### Q5: 呼吸练习可以自定义时长吗？
**A**: 当前版本固定使用4-7-8呼吸法。V1.1版本会增加更多呼吸法选择（方形呼吸、深腹式呼吸等）。

### Q6: 应用占用存储空间多少？
**A**: 
- 应用本身：约20-30MB
- 数据存储：每条记录约1KB
- 1000条记录约1MB
- 总体占用很小

### Q7: 如何完全删除所有数据？
**A**: 
- 方法1：设置 → 清除所有数据
- 方法2：系统设置 → 应用管理 → 清除数据
- 方法3：卸载应用

### Q8: 应用需要联网吗？
**A**: 不需要。应用完全离线运行，所有功能都可在无网络环境下使用。
